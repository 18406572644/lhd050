## 1. 产品概述

手工香材称量与配比模拟器是一款面向手工调香爱好者的纯前端工具应用，以极简实验室视觉风格呈现，模拟天平、量勺、器皿等器具交互，帮助用户完成香材录入、精准配比调试、配方管理与历史回溯的完整调香流程。

- 解决手工调香中配比计算繁琐、配方难以系统管理、历史记录不便追溯的痛点
- 目标用户为手工香爱好者、独立调香师、香道研习者

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 单用户 | 无需注册 | 全部功能 |

### 2.2 功能模块

1. **工作台页面**：天平称量模拟、量勺取料模拟、器皿盛放可视化、香材配比数值调试、配比误差提醒
2. **配方管理页面**：配方保存与命名、配比方案分类管理、常用配方一键套用、配方搜索与筛选
3. **历史记录页面**：配比历史时间线、历史记录详情回溯、历史配方对比

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 工作台 | 天平模拟区 | 可视化天平，随配比数值实时倾斜，左右托盘显示当前香材重量 |
| 工作台 | 量勺取料区 | 模拟不同规格量勺（1g/0.5g/0.1g），点击取料添加香材 |
| 工作台 | 器皿展示区 | 展示当前调配器皿中各香材层级，可视化堆积效果 |
| 工作台 | 配比数值面板 | 精确到0.01g的数值输入与滑块调整，支持增量/减量微调 |
| 工作台 | 误差提醒条 | 当配比总和偏离标准范围时显示警告，偏差超过5%标红提示 |
| 工作台 | 香材录入弹窗 | 添加自定义香材（名称、类型、单价、描述），支持预设香材库选择 |
| 配方管理 | 配方列表 | 按分类展示已保存配方，支持卡片/列表视图切换 |
| 配方管理 | 分类标签管理 | 自定义分类标签（如"花香型"、"木香型"、"合香"等），支持增删改 |
| 配方管理 | 配方编辑 | 修改配方名称、配比参数、备注，实时预览配比饼图 |
| 配方管理 | 一键套用 | 选中常用配方后一键加载至工作台，自动填充所有配比参数 |
| 配方管理 | 配方搜索 | 按名称、分类、香材关键词搜索配方 |
| 历史记录 | 时间线列表 | 按日期倒序展示调配历史，每条记录显示配方名、时间、状态 |
| 历史记录 | 详情回溯 | 展开历史记录查看完整配比详情，可一键恢复至工作台 |
| 历史记录 | 配方对比 | 选择两条历史记录进行配比差异对比 |

## 3. 核心流程

用户打开工作台，从预设香材库选择或自定义录入香材，通过量勺模拟点击或数值微调设定各香材用量，天平可视化实时反馈称量状态，器皿展示香材堆积效果。配比完成后可保存为命名配方并归入分类，也可一键套用已有配方。系统自动记录每次调配操作至历史，支持随时回溯与对比。当配比偏差超过阈值时自动弹出误差提醒。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：浅灰(#E8E4E0)、米白(#F5F1EB)、浅咖(#C4A882)
- **辅助色**：深咖(#8B6F4E)用于文字强调，警示橙(#D4915E)用于误差提醒
- **按钮风格**：圆角微弧，浅咖色主按钮，米白底灰色边框次级按钮
- **字体**：标题使用衬线体思源宋体（Noto Serif SC），正文使用无衬线思源黑体（Noto Sans SC）
- **布局风格**：左侧导航 + 右侧内容区，卡片式布局，大量留白营造实验室简洁感
- **图标风格**：线性手绘风格，贴合手工调香主题

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 工作台 | 天平模拟区 | CSS绘制的对称天平图示，左右托盘随重量倾斜动画，浅灰底+深咖线条 |
| 工作台 | 量勺取料区 | 三种规格量勺图标按钮，悬浮提示规格，点击时有下落微动画 |
| 工作台 | 器皿展示区 | 圆形容器剖面图，香材分层堆叠，不同香材以渐变色区分 |
| 工作台 | 配比数值面板 | 每种香材一行：名称+数值输入框+滑块+增减按钮，浅咖色高亮当前编辑行 |
| 工作台 | 误差提醒条 | 顶部固定提示条，绿色正常/黄色警告/红色超标，带动画过渡 |
| 工作台 | 香材录入弹窗 | 模态弹窗，表单含名称、类型下拉、单价、描述，左侧预设香材快速选择列表 |
| 配方管理 | 配方列表 | 卡片网格布局，每张卡片显示配方名、分类标签、配比概览迷你饼图 |
| 配方管理 | 分类标签 | 顶部横向标签栏，浅咖底白字选中态，支持内联编辑标签名 |
| 配方管理 | 配方编辑 | 侧滑面板，含配方基本信息+配比列表+实时饼图预览 |
| 历史记录 | 时间线 | 左侧竖向时间轴线，节点展示日期，右侧卡片展示调配摘要 |

### 4.3 响应式

- 桌面优先设计，最小宽度1024px
- 平板适配：工作台量勺区与器皿区纵向排列
- 移动端暂不做强制适配，保持核心功能可用

### 4.4 3D场景指引

不适用，本项目采用2D CSS/SVG视觉模拟方案。
